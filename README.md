# AI Voice English Tutor

Voice-first AI assistant for practicing spoken English.

## Features

- Voice input (speech-to-text)
- AI conversation
- Grammar corrections
- Voice responses

## Tech Stack

- Frontend: React + Vite
- Backend: Node.js + Express
- LLM: Llama 3
- Speech: Browser Web APIs
